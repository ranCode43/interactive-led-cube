# CAD Files

This folder contains the CAD files for the Interactive LED Cube.

## Files

- `cube_frame.f3d` – Original Fusion 360 project file.

- `cube_frame.step` – Universal CAD file for manufacturing and replication.

## Description

The cube is designed to be  100 mm × 100 mm × 100 mm.

Features include:

- Openings for RGB LED matrices on four faces

- Internal mounting space for the ESP32

- Mounting space for sensors such as the MPU6050 and microphone

- Access opening for USB programming and power

## Software Used

- Fusion 360
